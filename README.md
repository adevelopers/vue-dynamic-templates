# vue-dynamic-templates
VueJS: several templates for one component  


## Initial vue project
```
# install node
npm install

# install Vue(cli)
npm install -g @vue/cli

# get vue version
vue --version

# init project
vue create [project-name]

# run dev project
yarn serve

```

### Attention
In branch `vue-scripts` not used `vue-cli` + node, just `public` dir.
